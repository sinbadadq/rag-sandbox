# trace

Decorator for automatic function tracing with input/output capture.

Uses the global FiddlerClient when client= is not passed. Supports async functions with proper trace isolation per task.

## Parameters

| Parameter        | Type                                             | Required | Default | Description                                        |
| ---------------- | ------------------------------------------------ | -------- | ------- | -------------------------------------------------- |
| `func`           | \`Callable                                       | None\`   | ‚úó       | `None`                                             |
| `name`           | \`str                                            | None\`   | ‚úó       | `function name`                                    |
| `as_type`        | `Literal['span', 'generation', 'chain', 'tool']` | ‚úó        | `span`  | Span type ('span', 'generation', 'chain', 'tool'). |
| `capture_input`  | `bool`                                           | ‚úó        | `True`  | Capture function arguments as input.               |
| `capture_output` | `bool`                                           | ‚úó        | `True`  | Capture return value as output.                    |
| `client`         | \`FiddlerClient                                  | None\`   | ‚úó       | `get_client()`                                     |
| `model`          | \`str                                            | None\`   | ‚úó       | `None`                                             |
| `user_id`        | \`str                                            | None\`   | ‚úó       | `None`                                             |
| `version`        | \`str                                            | None\`   | ‚úó       | `None`                                             |
| `system`         | \`str                                            | None\`   | ‚úó       | `None`                                             |
